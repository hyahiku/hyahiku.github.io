## Projects

### 🌎 Climate Profile Segmentation & Retrieval
> A hybrid approach to analyzing municipal climate patterns across Brazil

This project applies K-Means clustering to segment over 5,500 Brazilian municipalities into four distinct climate profiles based on temperature, humidity, precipitation, and solar radiation. A RAG-inspired hybrid retrieval system was then built to allow users to query municipalities using natural language — combining structured filtering with semantic ranking for accurate multi-attribute results.

**Key highlights:**
- Cleaned and aggregated ~3.5 million weekly climate observations to the municipality level
- Evaluated K-Means solutions using silhouette scoring; selected k=4 for interpretability
- Identified four profiles: *Hot and Dry*, *Hot, Humid, and Rainy*, *Cool, Humid, and Variable*, and *Moderate and Balanced*
- Demonstrated that semantic-only retrieval is insufficient for structured environmental data; hybrid approach significantly improved result alignment

**Tech stack:**
`Python` `pandas` `scikit-learn` `sentence-transformers` `KMeans` `StandardScaler` `cosine similarity` `matplotlib` `plotly` `Hugging Face Datasets`
